## Что не отправляется в GitHub

- токены Telegram;
- пароли;
- API-ключи;
- `.env`;
- папки `credentials`;
- приватные SSH-ключи;
- журналы работы;
- локальные базы данных;
- полные дампы сервера;
- зашифрованные локальные файлы `*.dpapi`.
