# Диагностика МБП вар2

Статическое приложение для диагностики потребности в управленческом учете по файлу `анкета для собственника вар2.docx`.

## Запуск

Откройте файл `index.html` в браузере.

## Логика

- вопросы 1-6 формируют профиль бизнеса;
- вопросы 7-18 дают баллы риска по деньгам, прибыли и отчетности;
- вопросы 19-20 фиксируют приоритет и желаемый результат;
- вопрос 15 не влияет на баллы, если в бизнесе одно направление;
- итог показывает уровень риска, рекомендации по отчетам и следующий шаг.

Данные не отправляются на сервер и остаются в браузере.
