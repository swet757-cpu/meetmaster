# Spanish Bot

Локальный Telegram-бот для изучения испанских слов без VPS, n8n и внешнего планировщика.

## Как запустить

1. Создайте бота через `@BotFather` и получите токен.
2. Установите зависимости:

```powershell
pip install -r requirements.txt
```

3. Укажите токен в переменной окружения:

```powershell
$env:TELEGRAM_BOT_TOKEN="ВАШ_ТОКЕН"
```

4. Запустите бота:

```powershell
python main.py
```

## Команды в Telegram

```text
/start - краткая справка
/study - показать слово для повторения
/stats - статистика
```

## Как устроено повторение

Если нажать `Помню`, интервал растет по шагам: 1, 3, 7, 14, 30 дней.

Если нажать `Забыл`, интервал сбрасывается до 1 дня.

Файл `words.db` создается автоматически при первом запуске.
