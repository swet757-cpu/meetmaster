# Резервная копия программы

Дата: 2026-06-02.

Это копия файлов локального Telegram-бота без токена, журнала и служебного состояния.

Для восстановления скопировать файлы в папку `local_cash_flow_bot`, затем запустить `SETUP_AND_START.cmd` и ввести актуальный токен Telegram локально.

Токен в резервную копию не включен.
