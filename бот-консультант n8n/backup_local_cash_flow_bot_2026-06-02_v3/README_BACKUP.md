# Резервная копия локального бота

Дата: 2026-06-02.

Это копия программы `@cash_gap_buster_bot` без секретов.

В резервную копию не включены:

- токен Telegram;
- папка `credentials`;
- журнал работы;
- служебное состояние;
- временные файлы Python.

## Восстановление

1. Скопировать файлы в отдельную рабочую папку.
2. Запустить `ЗАПУСТИТЬ_СПАСАТЕЛЯ_ДЕНЕЖНОГО_ПОТОКА.cmd`.
3. Ввести актуальный токен Telegram из BotFather локально.

Токен нельзя отправлять в чат, почту или скриншоты.
