# Спасатель денежного потока: рабочий локальный запуск

Дата проверки: 2026-06-02.

## Результат

Telegram-бот `@cash_gap_buster_bot` работает локально на компьютере без VPS, без n8n Cloud и без платного AI API.

| Проверка | Результат |
|---|---|
| Токен Telegram настроен | да |
| Токен хранится обычным текстом | нет |
| Шифрование токена средствами Windows | да |
| Автозапуск после входа в Windows | включен |
| Успешный запуск бота | подтвержден |
| VPS | не нужен |
| n8n | не нужен |
| Supabase | не нужен |
| Платный AI API | не нужен |

## Как работает

Бот отвечает по готовой финансовой базе знаний:

- кассовый разрыв;
- прибыль есть, а денег нет;
- платежный календарь;
- короткие ответы;
- ссылки на материалы Google Drive.

## Условия бесплатной работы

Компьютер должен быть включен, не находиться в спящем режиме и иметь доступ в интернет.

После входа в Windows бот запускается автоматически.

## Файлы

Главный файл запуска:

`ЗАПУСТИТЬ_СПАСАТЕЛЯ_ДЕНЕЖНОГО_ПОТОКА.cmd`

Папка программы:

`бот-консультант n8n/local_cash_flow_bot`

Зашифрованный токен:

`бот-консультант n8n/local_cash_flow_bot/credentials/telegram_token.dpapi`

## Безопасность

Токен Telegram нельзя отправлять в чат, почту или скриншоты.

Если токен случайно отправлен в чат, его нужно сменить через `@BotFather` командой `/revoke`, затем повторно запустить файл настройки.

Зашифрованный файл токена не нужно копировать на другой компьютер: он привязан к текущему пользователю Windows.
