# Спасатель денежного потока: бесплатный локальный запуск

Бот работает без VPS, без n8n Cloud и без платного AI API.

Он отвечает по готовой финансовой базе знаний, показывает кнопки и дает ссылки:

- чек-лист при кассовом разрыве;
- гайд «прибыль и деньги»;
- шаблон платежного календаря.

## Первый запуск

1. Запустить двойным щелчком `SETUP_AND_START.cmd`.
2. Ввести токен Telegram-бота из BotFather. Токен вводится скрыто.
3. Открыть Telegram и отправить боту `/start`.

Токен хранится в зашифрованном виде средствами Windows. Он доступен только текущему пользователю Windows.

## Автозапуск

Автозапуск включается автоматически во время первого запуска. Бот будет запускаться после входа в Windows.

Для работы ботa компьютер должен быть включен и подключен к интернету. Если компьютер выключен или спит, бесплатный локальный бот не отвечает.

## Проверка

Отправить боту:

1. `/start`
2. `кассовый разрыв`
3. `прибыль есть, а денег нет`
4. `как вести платежный календарь`

## Остановка

Выполнить `stop_bot.ps1`.

Чтобы убрать автоматический запуск, выполнить:

`remove_autostart.ps1`

## Безопасность

Не отправлять токен Telegram в чат, почту или скриншоты.

В папке `credentials` лежит зашифрованный файл. Не отправлять эту папку другим людям.
