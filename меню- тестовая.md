## Безопасность

| Что | Правило |
|---|---|
| Telegram token | Не отправлять в чат и не показывать на скриншотах |
| OpenRouter API key | Не отправлять в чат и не показывать на скриншотах |
| Если ключ попал на скрин | Удалить старый ключ и создать новый |
| Где хранить ключи | Только в credentials n8n или переменных окружения |
