# Диагностика продвинутая для МБП

Статическое интерактивное коммерческое предложение для продажи услуг финдиректора на аутсорсе.

## Запуск

Откройте `index.html` в браузере.

## Что делает

- собирает профиль бизнеса;
- считает баллы финансового риска;
- показывает ориентир потерь по формуле;
- формирует выводы 5 экспертных логик;
- готовит текст КП для клиента.

## Без сервера

Данные остаются в браузере. Результат можно:

- скопировать кнопкой;
- распечатать или сохранить в PDF;
- отправить через Telegram;
- отправить через почту.
